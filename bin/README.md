# bin

Bu klasor, repo kokundeki yardimci scriptleri tutar.
Bunlar workspace geneline hizmet eden operasyon yardimcilari icindir.

## Ne zaman okunur?
- bir scriptin ne yaptigini anlamak icin
- workspace geneli otomasyon veya senkron mantigi incelenecekse
- operasyon akisini komut seviyesinde kontrol etmek icin

## Ilk bakilacak yerler
- `esnafdijital-workspace-sync`

## Yazma kurallari
- script eklemeden once mevcut script isi cozer mi kontrol et
- uygulama-ozel scriptleri repo koku yerine ilgili uygulama altinda tut
- yikici veya gizli bilgi isteyen akislari belge olmadan ekleme

## Not
Bu klasor repo geneli icindir. Uygulama-ozel scriptler icin `agent-workspace/bin/` ayri dusunulmelidir.
