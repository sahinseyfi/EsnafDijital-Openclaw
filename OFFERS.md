# OFFERS.md

## 1) Giriş Ürünü
### Dijital Görünürlük Kontrolü / Audit
Amaç: işletmenin dijitalde nasıl göründüğünü hızlıca ortaya çıkarmak.

Kapsam:
- harita varlığı kontrolü
- temel bilgi tutarlılığı kontrolü
- yorum / puan görünümü
- web sitesi var mı / güven veriyor mu
- iletişim kanalları görünür mü
- kısa dijital audit özeti

Teslim:
- kısa puanlama
- eksik listesi
- hızlı iyileştirme önerileri

Hariçler:
- detaylı SEO danışmanlığı
- kapsamlı sosyal medya yönetimi
- özel yazılım geliştirme

## 2) Ana Ürün
### Güven Veren Dijital Vitrin
Amaç: işletmenin dijitalde güven veren temel varlığını hızlıca kurmak.

Varsayılan kapsam:
- tek sayfa profesyonel web sitesi
- temel harita / profil düzeni
- QR yorum yönlendirme akışı
- temel güven unsurları (iletişim, konum, hizmet, görsel düzen, aksiyon butonları)
- hızlı teslim

Teslim sözü:
- sade
- hızlı
- işletmenin güven algısını artıran görünür iyileştirme

Hariçler:
- çok sayfalı kurumsal site
- özel panel geliştirme
- ileri entegrasyonlar

## 3) Devam Ürünü
### Bakım / Canlılık / Küçük Güncelleme
Amaç: kurulan vitrinin ölü kalmamasını sağlamak.

Varsayılan kapsam:
- küçük içerik güncellemeleri
- iletişim / saat / hizmet bilgisi değişiklikleri
- temel görünürlük kontrolü
- yorum akışı takibi
