# Layout System

## Layout sistemi
- container max width: `1200px`
- desktop content width: `1140px`
- tablet padding: `24px`
- mobile padding: `16px`

## 8pt spacing
- `4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80`

## Sayfa ritmi
- section arası: `64px–96px`
- kart içi padding: `20px / 24px`
- form alan arası: `16px`
- başlık ile paragraf arası: `12px`
- başlık ile kart grubu arası: `24px`

## Responsive mantık
- `sm 640+`: formlar rahatlar
- `md 768+`: 2 kolonlu düzen başlar
- `lg 1024+`: dashboard sidebar sabit olabilir
- `xl 1280+`: landing container oturur
