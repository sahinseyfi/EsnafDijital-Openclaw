# website

Bu klasor, EsnafDigital'in public-facing web vitrini icindir.
Su an hafif tutulur; ana amac tanitim, guven ve erisim katmanidir.

## Ne zaman okunur?
- landing veya vitrin sayfasi degisecekse
- mesajlasma, guven unsuru veya hizmet anlatimi guncellenecekse
- ana domain deneyimi incelenecekse

## Ilk bakilacak yerler
- `index.html` -> mevcut vitrin girisi

## Yazma kurallari
- burada ic operasyon mantigi veya admin akislarini buyutme
- satisa ve guvene hizmet etmeyen gereksiz ekran karmasasi ekleme
- public-facing icerik ile agent/admin iceriğini karistirma

## Not
Daha uygulama-ic operasyon agir isler icin `agent-workspace/` tarafina gidilir.
